# sbmgr
Biblioteka w Rust słuząca do obsługi zmiennych UEFI (Secure Boot).
Skrypty demonstracyjne w folderze src/demo.


Projekt rozwijany w związku z pracą magisterską na Uniwersytecie Marii Curie-Skłodowskiej w Lublinie. 
**Uruchamianie skryptów, z uwagi na mozliwosc uszkodzenia bazy danych UEFI zalecane jedynie w środowiskach wirtualnych**. W celu podjęcia realnych czynności administracyjnych skorzystaj z dojrzałych projektów, np. [sbctl](https://github.com/foxboron/sbctl).
